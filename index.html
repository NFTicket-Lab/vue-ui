# What is vue-polkadot?

It is a collection of Vue component libraries that allows for easier use of the Polkadot JS infrastructure inside your apps.

Getting Started

We also created step-by-step tutorials for zero hassle. Jump right in and get working your project with using vue-polkadot components.

# Available packages

This repository split up into a number of internal packages, namely -

  • vue-identicon - Display of identity icons based on theme as Vue component
  • vue-keyring - A keyring with browser-specific extensions
  • vue-settings - Setting management and defaults
